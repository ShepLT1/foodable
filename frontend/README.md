# Foodable Web App - Frontend

The frontend is built with **Vite**, **React**, and **TypeScript**.

## Node Version Setup

This project uses **Node.js 22.17.1**.

1. Install `nvm` by following the [nvm installation guide](https://github.com/nvm-sh/nvm#installing-and-updating).
2. From the `frontend/` directory, run:

```bash
nvm install       # One-time setup
nvm use           # Uses the version specified in .nvmrc
```

## Install Dependencies

```bash
npm install
```

## Run the Development Server

```bash
npm run dev
```

The development server will start at:

```text
http://localhost:5173
```

## Create a Production Build

```bash
npm run build
```

## Preview the Production Build

```bash
npm run preview
```
